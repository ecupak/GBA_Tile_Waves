# GBA_Tile_Waves
An experiment in GBA development.

Based on the tonc tutorial
https://www.coranac.com/tonc/text/toc.htm

# What it do
Spawns 2 waves and plays a sound every time.
When waves overlap, their color is blended - kind of. It comes from a precomputed palette.

Press L to enter a test mode where the waves always spawn on top of each other to see the overlap colors.
Press R to enter back to the normal demo.
